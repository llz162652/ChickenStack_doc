# 关于 ChickenStack

::: info
ChickenStack 是一个基于栈的图灵完备编程语言，受到 Brainfuck 启发但更加人类友好。
:::

## 项目简介

ChickenStack 旨在提供一种简单、优雅、强大的编程语言，让开发者能够：
- 理解栈数据结构的工作原理
- 学习逆波兰表达式（后缀表达式）
- 探索编程语言的设计和实现
- 享受编程的乐趣

## 设计理念

### 简单
- 使用直观的符号，易于记忆
- 语法简洁，学习曲线平缓
- 无需复杂的配置

### 优雅
- 基于栈的清晰逻辑
- 代码简洁美观
- 符合数学表达习惯

### 强大
- 图灵完备，可以计算任何可计算的函数
- 支持数学运算、循环、逻辑判断
- 提供完整的 Python API

## 技术栈

### 核心组件
- **虚拟机 (VM)**: 负责执行所有指令
- **解析器 (Parser)**: 将源代码转换为指令序列
- **IO 处理器**: 跨平台的输入输出处理
- **解释器**: 整合所有组件，提供完整的执行环境

### 开发语言
- **Python 3.7+**: 主要实现语言
- **VitePress**: 文档系统
- **Vite**: 开发工具链
