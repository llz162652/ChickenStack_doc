# 更新日志

## [未发布] v1.0.0-alpha

### 🎉 新功能

#### 核心语言特性
- **基于栈的编程语言**：采用逆波兰表达式（RPN）语法，所有操作都在栈上进行
- **图灵完备**：支持数学运算、循环、逻辑判断等，可以计算任何可计算的函数
- **人类可读性**：使用直观的符号，比 Brainfuck 更人类友好，易于理解和调试

#### 指令集
- **数学运算**：支持加法 (+)、减法 (-)、乘法 (*)、除法 (/)、取余 (%)
- **栈操作**：支持复制 (:)、交换 (\)、丢弃 ($)
- **逻辑运算**：支持相等 (=)、大于 (>)
- **输入输出**：支持打印数字 (.)、打印字符 (")、输入数字 (,)、输入字符 (?)
- **循环控制**：支持循环开始 ([)、循环结束 (])

#### Python API
- **ChickenStackVM**：虚拟机核心，负责执行所有指令
- **Parser**：代码解析器，将源代码转换为 Token 列表
- **IOHandler**：输入输出处理器，支持跨平台兼容
- **ChickenStackInterpreter**：高级解释器，整合所有组件

#### 运行模式
- **文件模式**：支持从文件读取源代码并执行
- **演示模式**：内置示例代码，展示语言功能

#### 文档系统
- **VitePress 文档**：完整的在线文档系统
- **入门指南**：详细介绍语言基础和概念
- **API 文档**：完整的 Python API 参考手册
- **示例代码**：丰富的示例程序，包括基础示例、进阶示例和其他示例

### ✨ 功能增强

#### 跨平台支持
- **Windows 平台**：自动设置 UTF-8 编码输出，解决控制台中文显示问题
- **Linux/macOS 平台**：完美支持，无需额外配置

#### 错误处理
- **文件不存在错误**：友好的错误提示
- **读取文件错误**：详细的错误信息
- **运行时错误**：捕获所有异常，打印详细错误信息，包括：
  - 错误位置（Token 编号、行号、列号）
  - 当前栈状态
  - 已执行迭代次数
  - 执行时间

#### 性能优化
- **迭代计数器**：防止无限循环
- **跳转验证**：防止索引越界
- **执行时间统计**：显示代码执行时间
- **指令映射字典**：使用字典映射执行操作，提高执行效率

#### 调试功能
- **栈状态查看**：随时查看当前栈状态
- **执行统计**：显示指令数、迭代次数、执行时间
- **错误详情**：详细的错误信息，便于调试
