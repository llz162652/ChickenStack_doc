## 详细说明

### ASCII 码

ChickenStack 使用 ASCII 码来表示字符：

| 字符 | ASCII 码 |
|------|----------|
| H | 72 |
| E | 69 |
| L | 76 |
| L | 76 |
| O | 79 |
| 换行 | 10 |

### 指令说明

- `数字`: 推入数字到栈
- `"`: 打印栈顶数字对应的 ASCII 字符
- `10`: 换行符的 ASCII 码

### 执行过程

```
72 "  → 推入 72，打印 'H'
69 "  → 推入 69，打印 'E'
76 "  → 推入 76，打印 'L'
76 "  → 推入 76，打印 'L'
79 "  → 推入 79，打印 'O'
10 "  → 推入 10，打印换行
```
