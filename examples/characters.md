## 使用 Python API

### 示例1: 基本字符输出

```python
from chicken_stack import ChickenStackVM

vm = ChickenStackVM()

# 打印字符
vm.push(72)
vm.op_print_char()  # H

vm.push(101)
vm.op_print_char()  # e

vm.push(108)
vm.op_print_char()  # l

vm.push(108)
vm.op_print_char()  # l

vm.push(111)
vm.op_print_char()  # o

vm.push(10)
vm.op_print_char()  # 换行
```

### 示例2: 字符串列表

```python
from chicken_stack import ChickenStackVM

vm = ChickenStackVM()

# 使用列表打印字符串
message = "Hello"
for char in message:
    vm.push(ord(char))
    vm.op_print_char()

vm.push(10)
vm.op_print_char()
```

### 示例3: ASCII 码转换

```python
from chicken_stack import ChickenStackVM

vm = ChickenStackVM()

# 字符转 ASCII
char = 'A'
ascii_code = ord(char)
print(f"{char} = {ascii_code}")  # A = 65

# ASCII 转字符
ascii_code = 65
char = chr(ascii_code)
print(f"{ascii_code} = {char}")  # 65 = A
```

### 示例4: 自定义打印函数

```python
from chicken_stack import ChickenStackVM

class PrintVM(ChickenStackVM):
    def print_string(self, s):
        """打印字符串"""
        for char in s:
            self.push(ord(char))
            self.op_print_char()

# 使用自定义打印
vm = PrintVM()
vm.print_string("Hello World")
```
